# WHAT IS IT?
# This bash script is used to write the numbers from 1 to 10 in a random order.

# HOW TO USE IT?
# Download the file, give permissions to execute (chmod +x "path_to_the_file") and execute with "/path_to_the_file/exerc1.sh"
